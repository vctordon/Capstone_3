# Capstone_3

This project is an interactive Power BI dashboard that is being used to analyze sales performance for the Northern Region of Emporium.

Objectives: 
- Analyze overall sales performance in the Northeast region
- Identify top-performing states, stores, and product categories
- Evaluate sales trends over time
- Provide actionable business recommendations

The Power BI report includes

- Line chart showing sales trends over the full period covered in the data
- Column or bar chart showing sales performance by category
- Pie or donut chart showing relative sales for each state in the region
- List of top-selling books, including author names, for your region (general audience,
excluding textbooks)
